# 🎙️ J.A.R.V.I.S for Web

**Just A Rather Very Intelligent System for your browser.**

![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)
![Version](https://img.shields.io/badge/version-1.0.0-blue)

JARVIS-Web is a next-generation browser extension that brings the power of AI and the Model Context Protocol (MCP) directly into your browsing experience. With an Iron Man-inspired HUD, it acts as your personal digital butler.

## ✨ Features

- **🛡️ Integrated AI**: Connect your OpenAI or Anthropic keys to enable cognitive capabilities.
- **🔌 MCP Ready**: Designed to integrate with Model Context Protocol servers for browser automation.
- **🌌 Futuristic HUD**: Glassmorphism UI with glow effects and hexagon grid overlays.
- **⌨️ Instant Access**: Use `Ctrl+Shift+J` (or `Cmd+Shift+J`) to activate JARVIS on any page.
- **📜 Side Panel Support**: Run JARVIS alongside your content for seamless multitasking.

## 🛠️ Installation

1. **Download** or clone this repository.
2. Open your browser's **Extensions** page:
   - Chrome/Edge: `chrome://extensions`
   - Brave: `brave://extensions`
3. Enable **Developer Mode** (top right toggle).
4. Click **Load unpacked** and select the `JARVIS-Web` folder.
5. Click the J.A.R.V.I.S icon or use the shortcut to begin.

## ⚙️ Configuration

1. Open the JARVIS popup.
2. Click **SETTINGS** at the bottom.
3. Paste your AI API Key (OpenAI or Anthropic).
4. Select your preferred model.
5. Close settings and start giving commands!

## 🚀 Usage

- **"Summarize this page"**: JARVIS will read the current tab and give you a breakdown.
- **"Help me write an email"**: Use the AI engine to draft content.
- **"Analyze this site"**: Leverages browser APIs to give you insights into the current domain.
